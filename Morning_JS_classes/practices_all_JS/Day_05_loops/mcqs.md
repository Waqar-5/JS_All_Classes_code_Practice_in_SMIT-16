## 18. For Loops (30 MCQs)

1. What is the syntax of a basic for loop?

   * a) for(initialization; condition; increment) {}
   * b) for(condition; initialization; increment) {}
   * c) for(increment; condition; initialization) {}
   * d) for(initialization; increment; condition) {}

2. What is executed first in a for loop?

   * b) Increment
   * c) Initialization
   * d) Loop body

3. When is the condition checked?

   * b) After each iteration
   * c) Only once

4. What happens if the condition is always true?

   * b) Infinite loop
   * c) Loop executes once

5. What does `for(let i=0;i<5;i++)` print?

   * a) 0 to 4
   * b) 1 to 5
   * c) 0 to 5
   * d) 1 to 4

6. Can `for` loops decrement?

   * a) Yes, by using i--

   * d) Only in nested loops

7. What does `i+=2` do in a loop?

   * a) Increment by 1
   * b) Increment by 2
   * c) Decrement by 2
   * d) No change

8. Which part of loop is optional?

   * a) Initialization
   * b) Condition
   * c) Increment
   * d) All parts

9. Loop variable scope if declared with `let`?

   * a) Global
   * b) Block-scoped
   * c) Function-scoped
   * d) None

10. Loop variable scope if declared with `var`?

    * a) Global
    * b) Function-scoped
    * c) Block-scoped
    * d) None

11. Can multiple variables be initialized in a for loop?

    * a) Yes, separated by commas
    * b) No

12. Can multiple increments be done in a for loop?

    * a) Yes, separated by commas
    * b) No

13. Is `for(;;)` valid?

    * a) Yes, infinite loop
    * b) No

14. Can you leave increment empty?

    * a) Yes
    * b) No

15. Can you leave initialization empty?

    * a) Yes
    * b) No

16. Can you leave condition empty?

    * a) Yes, creates infinite loop
    * b) No

17. What does `for(let i=10;i>0;i--)` print first iteration?

    * a) 10
    * b) 0
    * c) 1
    * d) 9

18. Can for loops iterate over arrays?

    * a) Yes
    * b) No

19. Can you use break to exit a for loop?

    * a) Yes
    * b) No

20. Can you use continue in for loop?

    * a) Yes
    * b) No

21. Loop executes zero times if?

    * a) Condition false initially
    * b) Condition true initially
    * c) No increment
    * d) Using continue

22. Which keyword defines block scope for loop variable?

    * a) var
    * b) let
    * c) const
    * d) function

23. Can loop variable be constant?

    * a) No, const cannot change
    * b) Yes

24. Can loop variable be updated inside loop body?

    * a) Yes
    * b) No

25. For loop vs while loop: main difference?

    * a) Syntax and initialization
    * b) Functionality
    * c) Arrays only
    * d) Cannot decrement

26. Infinite loop syntax?

    * a) for(;;) {}
    * b) for(i=0;i<10;i++) {}
    * c) while(i<10) {}
    * d) for(i=0;;) {}

27. Is semicolon at end of for loop mandatory?

    * a) No, if block present
    * b) Yes

28. Can you declare loop variable outside?

    * a) Yes
    * b) No

29. Which part runs after each iteration?

    * a) Initialization
    * b) Condition
    * c) Increment/decrement

30. Typical use case of for loop?

    * a) Repeating code known number of times

## 19. For Loops: Flags, Booleans, Array Length, and Breaks (30 MCQs)

1. What is a flag in programming?

   * a) Boolean variable
   * b) Array
   * c) Loop
   * d) Function

2. Which type stores true/false?

   * a) Number
   * b) Boolean
   * c) String
   * d) Object

3. What does break do in loop?

   * a) Stops loop
   * b) Skips iteration
   * c) Continues loop
   * d) Throws error

4. What does continue do in loop?

   * a) Skips iteration
   * b) Stops loop
   * c) Restarts program
   * d) Exits function

5. How to loop through array dynamically?

   * a) Use array.length
   * b) Hardcode size
   * c) Use while only
   * d) Use recursion

6. Flag initialized to false, set true if condition met, then break: purpose?

   * a) To detect a condition
   * b) To skip loop
   * c) To increment variable
   * d) To exit function

7. Can multiple flags be used in a loop?

   * a) Yes
   * b) No

8. What happens if break used in inner loop only?

   * a) Exits inner loop
   * b) Exits outer loop
   * c) Exits program
   * d) Skips iteration

9. Can continue be combined with flags?

   * a) Yes
   * b) No

10. Using array.length prevents?

    * a) Hardcoding indices
    * b) Infinite loops
    * c) Errors accessing array
    * d) All

11. Can break be used in nested loops?

    * a) Yes, affects only current loop
    * b) No

12. Is flag necessary for break?

    * a) No
    * b) Yes

13. Can you use multiple conditions with flags?

    * a) Yes
    * b) No

14. Boolean can be?

    * a) true or false
    * b) 0 or 1
    * c) Both a and b

15. Typical usage of flags in loops?

    * a) To indicate a search result
    * b) To sum numbers
    * c) To print
    * d) None

16. Can continue be used alone?

    * a) Yes
    * b) No

17. Difference between break and continue?

    * a) break exits loop, continue skips iteration
    * b) break skips iteration, continue exits loop

18. Does flag reset automatically each iteration?

    * a) No, manual reset needed
    * b) Yes

19. How to loop backward through array?

    * a) for(let i=arr.length-1;i>=0;i--)
    * b) for(let i=0;i<arr.length;i++)

20. Can you combine break and continue in same loop?

    * a) Yes
    * b) No

21. Flag initialized outside loop ensures?

    * a) It persists across iterations
    * b) Resets each time

22. Break inside if inside loop?

    * a) Exits loop if condition true
    * b) Skips iteration

23. Can boolean expression be used as condition?

    * a) Yes
    * b) No

24. For loop over empty array with length 0?

    * a) Loop does not execute
    * b) Loop executes once

25. Can flag hold multiple states?

    * a) No, boolean only two states

26. Can break stop infinite loop?

    * a) Yes
    * b) No

27. Example of using flag in loop?

    * a) Searching element in array
    * b) Printing numbers

28. Using array.length avoids?

    * a) Index out of bounds

29. Can continue affect outer loop in nested loop?

    * a) No
    * b) Yes

30. Best practice with flags?

    * a) Initialize outside loop, update inside

## 20. For Loops Nested (30 MCQs)

1. What is a nested loop?

   * a) Loop inside another loop
   * b) Two loops side by side
   * c) Function inside loop
   * d) Array inside loop

2. How many times does inner loop execute?

   * a) For each outer loop iteration
   * b) Once
   * c) Twice
   * d) Depends on condition

3. Typical use of nested loops?

   * a) 2D arrays, tables
   * b) Single variable increment
   * c) Event handling

4. Can break in inner loop affect outer loop?

   * a) No
   * b) Yes

5. Can continue in inner loop affect outer loop?

   * a) No
   * b) Yes

6. Syntax of nested loop?

   * a) for(...) { for(...) {} }
   * b) for(...) {} for(...) {}

7. Nested loops can be more than 2 levels?

   * a) Yes
   * b) No

8. Typical risk with nested loops?

   * a) Performance issues
   * b) Syntax errors

9. Example of nested loop with multiplication table?

   * a) for i 1-3, inner j 1-3, multiply

10. Outer loop variable changes per inner loop iteration?

    * a) No

11. Inner loop variable accessible outside inner loop?

    * a) No, if declared with let

12. Can nested loops be used for pattern printing?

    * a) Yes

13. Can break exit both loops?

    * a) Not directly, need flag or label

14. Example of labeled break?

    * a) outer: for(...) { for(...) { break outer; } }

15. Nested loop over array of arrays?

    * a) Yes

16. Can you use continue in outer loop based on inner loop?

    * a) Yes, with flag

17. Nested loop increments?

    * a) Inner loop increments independently

18. Can nested loops modify arrays?

    * a) Yes

19. Typical output of nested loop printing i,j?

    * a) i=1,j=1 ... i=3,j=3

20. Can nested loops be infinite?

    * a) Yes, if conditions always true

21. Can nested loops be used in DOM manipulation?

    * a) Yes

22. Outer loop iterates 3 times, inner 4 times: total iterations?

    * a) 12

23. Nested loops with different ranges?

    * a) Yes

24. Inner loop variable overrides outer loop variable?

    * a) Only if same name and scope allows

25. Best practice: keep variable names unique?

    * a) Yes

26. Can nested loops be used for grid layouts?

    * a) Yes

27. Can nested loops be combined with arrays and flags?

    * a) Yes

28. Performance tip for nested loops?

    * a) Minimize iterations where possible

29. Can nested loops call functions?

    * a) Yes

30. Exam tip for nested loops?

    * a) Track outer vs inner, watch break/continue
