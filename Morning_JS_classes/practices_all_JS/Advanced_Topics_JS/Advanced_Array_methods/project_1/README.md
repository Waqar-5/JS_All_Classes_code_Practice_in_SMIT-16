## Overview
**JS Brain Trainer & Data Explorer** is a **unique, interactive, multi-layer single-page project** that demonstrates mastery of **advanced JavaScript concepts**. It combines **mini games, dynamic data exploration, and a task manager** in one polished, portfolio-ready project.  

This project is ideal for showcasing your **deep JavaScript skills** to employers, on LinkedIn, or in a GitHub portfolio.

---

## Live Demo
You can run this project locally by opening the `index.html` file in a web browser.  
Alternatively, deploy it on GitHub Pages for a live demo.

---

## Features
- **Memory Game:** Match pairs using array methods, arrow functions, Map & Set  
- **Math Trainer:** Solve random math questions using destructuring, rest/spread, and template literals  
- **Data Explorer:** Generate and manipulate dynamic data with Map, Set, and advanced array methods  
- **Mini Task Manager:** Add, delete, toggle tasks; see live stats (total, completed, pending)  
- **Multi-Layer Navigation:** Easily switch between sections to explore different JS concepts  
- **Interactive & Visually Appealing:** Color-coded cards, dynamic messages, and responsive layout  

---

## Advanced JS Concepts Covered
- `var`, `let`, `const` variable types  
- Template literals for dynamic HTML and messages  
- Arrow functions for concise syntax and callbacks  
- Iterators (`for..of`) to loop through arrays, sets, and maps  
- Array Advanced Methods: `map`, `filter`, `reduce`, `sort`, `flatMap`, `find`, `findIndex`  
- Destructuring for objects and arrays  
- Rest & Spread operators for merging arrays and passing dynamic parameters  
- `Set` for unique collections  
- `Map` for key-value data management  
- Default parameters for flexible functions  
- JS Behind the Scenes: closures, IIFE, and private state handling
