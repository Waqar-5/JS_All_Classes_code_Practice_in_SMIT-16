**Answer:**

### Q2. Which symbol is used for the Rest Operator?
**Answer:**  
Three dots `...` are used to represent the rest operator.

### Q3. Is the Rest Operator an ES6 feature?
**Answer:**  
Yes. The rest operator was introduced in **ES6 (ECMAScript 2015)**.

### Q4. What does the Rest Operator return?
**Answer:**  
The rest operator **always returns an array**, never an object or single value.

**Answer:**

**Answer:**
