## Overview
This is a **single-file, interactive HTML app** that demonstrates:  
- Basic arrow functions  
- Arrow functions with multiple statements  
- Array operations using `map`, `filter`, and `reduce`  
- Lexical `this` in objects and callbacks  
- Nested and curried arrow functions  
- Rest parameters  

The app allows you to **input data and see live results**, making it perfect for **practical learning** and **interview preparation**.

## How to Use
1. Clone or download this repository.  
2. Open `index.html` in your browser.  
3. Interact with the different sections:  
   - **Calculator:** Input numbers and choose operations.  
   - **Array Playground:** Enter numbers separated by commas and click "Run Array Operations".  
   - **Lexical `this` Explorer:** Click the button to see team members with lexical `this`.  
   - **Nested Arrow Functions:** Input numbers to see curried multiplication in action.  
   - **Rest Parameters:** Enter numbers separated by commas to calculate their sum.  
4. Observe the live output and read the explanations.
