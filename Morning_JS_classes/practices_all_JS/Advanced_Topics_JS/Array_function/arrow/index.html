All About Arrow Functions

Arrow Function Overview

Arrow functions are a shorter syntax for writing JavaScript functions. They are anonymous and change how the `this` keyword behaves.

Example: 

const add = (a, b) => a + b;
console.log(add(2,3)); // 5

Features:

  • Shorter syntax
  • Implicit return for single-expression functions
  • Lexical `this` binding
