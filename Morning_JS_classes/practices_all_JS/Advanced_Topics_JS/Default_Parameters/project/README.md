# Dynamic Greeting Dashboard - Learn JavaScript Default Parameters

A hands-on project to **understand and practice default parameters in JavaScript**.  
This project uses **HTML, CSS, Bootstrap, and JS** to create a **dynamic greeting dashboard** that shows how **default parameters** work in real-world scenarios.

- Enter your **name**, **greeting type**, and **custom message**.  
- If any field is **left empty**, the app uses **default values**.  
- Demonstrates all types of default parameters:  
  - Constants (`"Guest"`)  
  - Expressions (dynamic greeting based on current hour)  
  - Functions (random motivational quote)  
  - Arrays (list of quotes)  

**Features:**  
1. Name defaults to `"Guest"` if empty  
2. Greeting type defaults based on **current time**  
3. Message defaults to `"Welcome to the Dashboard!"`  
4. Random motivational quote defaults from predefined array  
5. Real-time display using **DOM manipulation**
